Project Overview

The PA Saturation Simulation project models photoacoustic (PA) measurements and performs spectral unmixing to analyze tissue saturation levels at various wavelengths. This simulation simulates tissue properties by simulating the photoacoustic effect and processing the sensor data from a linear array of trasnducers.

Features

  • Simulates photoacoustic pressure waves in tissues with customizable absorption coefficients and concentration profiles.
  • Performs spectral unmixing to determine the relative concentrations of biomarkers like Hb and HbO.
  • Handles various noise levels and performs image reconstruction using k-Wave.
  • Generates visualizations for pressure waves, concentrations, and saturation maps.

Usage Instructions

To run the simulation, configure the input parameters, including tissue shape, absorption coefficients, and concentrations. The code will generate pressure wave data, which is then processed and reconstructed with different noise levels to simulate realistic measurement scenarios.

Getting Started

  1. Clone or download the repository.
  2. Set the input parameters in the configuration files located in the PA_Saturation directory.
  3. Run the MATLAB scripts to generate and process the simulation data.
  4. Analyze the results using the generated visualizations and spectral unmixing results.
