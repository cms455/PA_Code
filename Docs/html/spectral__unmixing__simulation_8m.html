Detailed Description

Performs complete simulation and unmixing for PA saturation.

Author
Calvin Smith
Date
12-20-24

Simulates a photoacoustic measurement and unmixing of tissues for one wavelength set.

  • Create an arbitrary shape/tissues and define the concentration of the different types of biomarkers.
  • Construct the pressure wave generated by a laser of wavelength using the formula: P(w, x, y) = Epsilon(w, t) * C(type, x, y).
    • P: Pressure
    • Epsilon: Absorption coefficients by wavelength and type
    • C: Concentration
  • Then measure using a transducer with spacing dx for Nx transducers (future versions will make this customizable).
  • Add different noise levels to the sensor data, and reconstruct the noisy sensor data.
  • Perform spectral unmixing on the reconstructed data and store the calculated concentrations and simulations.
Parameters
maskA boolean mask of size (Nx, Ny) that defines the shape of the tissue to simulate.
epsilonA (num_wavelength, num_types) matrix of absorption coefficients. Must match the order of concentrations.
wavelengthsA 1D array of wavelengths (e.g., [770, 780]). Must match the size of epsilon.
concentrationsAn array of size (unique_c*num_iter, num_types) defining the relative saturations of each type in the generated pressure mask.
type_namesA cell array of names (e.g., {'Hb', 'HbO'}). Order must match epsilon and concentrations.
noise_levelsA 1D array specifying the percentages of noise strength.
noise_strengthA scalar value for the maximum noise strength.
NxNumber of X pixels.
NyNumber of Y pixels.
dxSize of each X pixel in meters.
dySize of each Y pixel in meters.
plot_flag_pressuresBoolean flag. If true, plots each pressure (use for testing).
plot_flag_concentrationsBoolean flag to plot figures for each noise and concentration for all types and wavelengths.
plot_flag_saturationsBoolean flag to plot figures for each noise and saturation for all types and wavelengths.
save_flagBoolean flag. If true, saves specunmix_noise_c_data to the specified folder.
folder_pathThe folder path to save data. If it doesnât exist, it will be created.
Returns
specunmix_noise_c_data A cell array of shape {n, c, 4}. Each (n, c) coordinate stores:
  • sum_C: Total concentration maps.
  • saturations_by_type: Saturation maps by type.
  • concentrations_by_type: Concentration maps by type.
  • w_avg_by_type: Scalar average maps.
noisy_sensor_data_holder Holds all the noisy sensor data for testing to confirm appropriate noise levels.
