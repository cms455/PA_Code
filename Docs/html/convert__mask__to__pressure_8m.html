Detailed Description

Converts boolean mask, concentration set, and absorption coefficient matrix into pressure masks.

Author
Calvin Smith
Date
12-20-24

This function calculates pressure masks to be converted into sensor data. The pressure is calculated using the formula: P(w, x, y) = Epsilon(w, t) * C(type, x, y) where:

  • P: Pressure
  • Epsilon: Absorption coefficients by wavelength and type
  • C: Concentration
Parameters
maskA boolean mask of size (Nx, Ny) defining the shape of the tissue to simulate.
epsilonA (num_wavelength, num_types) matrix of absorption coefficients. Must match the order specified in concentrations.
wavelengthsA 1D array of wavelengths (e.g., [770, 780]). Must have the same size as epsilon.
concentrationsAn array of size (unique_c * num_iter, num_types) defining the relative saturations of each type in the generated pressure mask.
type_namesA cell array of names (e.g., {'Hb', 'HbO'}). The order must match with epsilon and concentrations.
NxNumber of X pixels.
NyNumber of Y pixels.
plotflagA boolean flag. If true, plots each pressure mask for all num_concentrations and wavelengths. Use only for testing when num_iter = 1.
Returns
pressure_mask The calculated pressure using P(w, x, y) = Epsilon(w, t) * C(type, x, y). This is used for calculating the sensor data.
