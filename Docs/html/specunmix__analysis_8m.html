Detailed Description

Analyzes full data (saturations, concentrations, weighted averages) from different wavelength sets.

Author
Calvin Smith
Date
12-20-24

This function performs analysis on spectral unmixing data, including:

  • Mean Saturation and Concentration:
    • Calculates mean and standard deviation for saturation by averaging over masked values (not the entire region for better spatial accuracy).
    • Plots mean saturation and concentration values with error bars, with:
      • X-axis: Noise level.
      • Y-axis: Measured quantity.
    • Generates separate figures for each noise level and unique concentration, with different lines for each wavelength.
  • Weighted Average Analysis:
    • Similar analysis for weighted averages (w_avg).
  • Difference Calculation:
    • Computes and plots a bar chart for the difference between mean saturation and concentration values.
Parameters
specunmix_dataCell array of specunmix_noise_c_data for multiple wavelengths. Each specunmix_noise_c_data has the structure {n, c, 4}, where each (n, c) contains:
  • sum_C: Total concentrations.
  • saturations_by_type: Saturations by type.
  • concentrations_by_type: Concentrations by type.
  • w_avg_by_type: Weighted averages by type.
maskBoolean tissue mask of size (Nx, Ny).
wavelength_setsArray concatenating wavelength sets (e.g., [750, 850; 770, 780]).
concentrationsArray of size (unique_c * num_iter, num_types), defining the relative saturations of each type in the generated pressure mask.
noise_simpleCombined noise levels and noise strength.
type_namesCell array of type names (e.g., {'Hb', 'HbO'}). Must match the order in epsilon and concentrations.
NxNumber of X pixels.
NyNumber of Y pixels.
save_flagBoolean flag indicating whether to save full_specunmix_analysis_data. Data includes:
  • saturation_mean_holder: Mean saturation values.
  • saturation_std_holder: Standard deviation of saturation values.
  • concentration_mean_holder: Mean concentration values.
  • concentration_std_holder: Standard deviation of concentration values.
  • w_avg_mean_holder: Mean weighted averages.
  • w_avg_std_holder: Standard deviation of weighted averages.
  • sat_difference_holder: Difference between mean saturation and concentration values.
num_iterNumber of iterations for each unique concentration.
folder_pathPath to the folder for saving analysis files.
plot_flag_cBoolean flag to plot mean concentrations with error bars for different wavelengths.
plot_flag_sBoolean flag to plot mean saturations with error bars for different wavelengths.
plot_flag_w_avgBoolean flag to plot mean weighted averages with error bars for different wavelengths.
plot_flag_diffBoolean flag to plot bar charts of the differences for different wavelengths.
Returns
full_specunmix_analysis_data Contains:
  • saturation_mean_holder: Mean saturation values.
  • saturation_std_holder: Standard deviation of saturation values.
  • concentration_mean_holder: Mean concentration values.
  • concentration_std_holder: Standard deviation of concentration values.
  • w_avg_mean_holder: Mean weighted averages.
  • w_avg_std_holder: Standard deviation of weighted averages.
  • sat_difference_holder: Difference between mean saturation and concentration values.
